Individual Balance Dashboard
============================

Run instructions:
1. Install Node.js (v16+ recommended).
2. Open terminal in this folder.
3. Run:
   npm install
   npm start
4. Open http://localhost:3000

Structure:
- server.js -> Express backend
- public/index.html + app.js + styles.css -> React frontend
- public/individual_balance_2021.json -> data exported from Excel
